# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- `CODE_OF_CONDUCT.md` (Contributor Covenant 2.1)
- `docs/github-growth-checklist.md` — GitHub metadata & discoverability guide
- Refreshed `CONTRIBUTING.md` / `CONTRIBUTING.en.md` with Good First Issues,
  local trial steps, and Discussions links

### Changed
- **Brand unification**: repository, README, docs, and marketing materials now

- **README rewrite** (English + 中文): accurate feature breadth — 62 slash
  commands, 20 analysis skills, 16 clinical specialty domains — with SEO- and
  GitHub-friendly structure, corrected badges, and a Table of Contents.

## [2.0.0] - 2026-05-10

### Added
- 20 specialized analysis skills (sleep, nutrition, fitness, mental health,
  TCM constitution, family, child health, women's/men's health, travel,
  occupational, rehabilitation, and more)
- Whole-lifecycle coverage: child health, pregnancy/postpartum, menopause,
  chronic disease (diabetes, hypertension, COPD) tracking
- Drug interaction database with five-level severity (A/B/C/D/X)

### Changed
- Expanded command set from the original core to 60+ slash commands
- Multi-Disciplinary Team (MDT) consultation across 16 specialty domains

## [1.0.0] - 2025-01-01

### Added
- Initial release: file-based personal health record system
- Core commands: profile, save-report, query, radiation, surgery, discharge,
  medication, allergy
- Basic specialist analysis and data privacy model
